# 认识一个人 · 渐进式协议清单

> 工程师视角的"建模一个人"指南：把模糊的人际过程，拆成有顺序、有字段、有解锁条件的状态机。
> 核心思路 —— 你在为对方逐层"反序列化"一个对象：**不能跳级加载，每一级要等"握手信号"确认，才进下一级。**

---

## 一、协议规则（5 条不变量，全程生效）

1. **顺序执行，禁止跳级。** 在 L1 阶段问 L4 的问题（"你最深的恐惧是啥"），对方会觉得被审讯。逐级来。
2. **双向，且你先 commit。** 每问一格，先交出你自己同一格的答案（"我是这样，那你呢？"）。单向提问 = 查户口。
3. **"听" = 追问，不是评价。** 收到回答，回一句"后来呢？""那你当时啥感觉？"；**不要**评判 / 反驳 / 急着给建议。
4. **一次一格，别批量。** 别把一级的问题一口气问完，自然地散在日常相处里。
5. **升级信号 = 对方主动回敬。** 你抛了，对方**也主动问回你 / 主动多讲一点 / 不用你问就分享** —— 这一级才算解锁。对方只敷衍、从不反问 = 这级还没过，别硬往深走。

> ⚠️ **禁区字段（pull-only）：** 创伤、前任、疾病、家庭伤痛。只能等对方主动 push 给你，**你绝不主动拉取**。

---

## 二、核心进阶清单（L0 → L5，照着勾）

### ☐ L0 · 基础档案（零风险，纯事实）
- [ ] 老家哪里
- [ ] 家里几口人 / 排行
- [ ] 现在住哪、一个人住还是合租
- [ ] 做什么工作 / 学什么
- [ ] 作息（早睡党 / 夜猫子）
- [ ] 有没有养宠物
- [ ] 爱吃什么 / 忌口什么
- [ ] 休息日一般干嘛
- **解锁信号：** 能轻松闲聊，对方也会问你这些。

### ☐ L1 · 当下的生活（低风险，开始 personal）
- [ ] 对方最近在忙什么
- [ ] 压力来自哪里
- [ ] 在期待 / 盼着什么事
- [ ] 最好的朋友是谁（名字）
- [ ] 最近最开心的一件事
- [ ] 最近最烦的一件事
- [ ] 喜不喜欢现在的工作 / 学业
- **解锁信号：** 对方会主动跟你讲"今天发生了啥"（报备型分享）。

### ☐ L2 · 喜好背后的"为什么"（关键过渡层）
- [ ] 对方为什么喜欢某部电影 / 某首歌 / 某个爱好
- [ ] 对方是怎么入这个坑的
- [ ] 什么东西能让对方"回血"（怎么放松、独处还是社交充电）
- [ ] 对方觉得什么好笑
- [ ] 什么事会让对方烦 / 反感
- **为什么重要：** "为什么"是把**数据**变成**理解**的那一步。L0/L1 是查参数，L2 才开始真正建模。
- **解锁信号：** 对方愿意展开讲"为什么"，而不是只甩一个结论。

### ☐ L3 · 来历与塑造（中高风险）
- [ ] 在哪长大、童年 / 家庭氛围
- [ ] 塑造过对方的关键经历
- [ ] 对方引以为傲的事
- [ ] 对方跟父母的关系
- ⚠️ **禁区：** 前任、家庭创伤——别主动挖。
- **解锁信号：** 对方开始**不设防地**讲过去，包括不那么光鲜的部分。

### ☐ L4 · 内心世界：恐惧 / 价值观 / 梦想（深层）
- [ ] 对方怕什么、担忧什么
- [ ] 对方真正在乎和相信的东西（价值观、世界观）
- [ ] 对方想要什么样的人生
- [ ] 对方的底线和原则
- **关键：** 这层最容易踩雷。任务是 **好奇 + 尊重**，不是认同，更不是反驳。**观点不同也能过关；贬低、嘲讽、拿弱点攻击 = 挂科。**
- **解锁信号：** 对方愿意把"软的、可能被人笑的"想法讲给你听——这是对方在赌你安全。

### ☐ L5 · 关系层：对方在感情里要什么（最深，需高信任）
- [ ] 对方怎样才感到被爱 / 被在乎
- [ ] 对方在过去的关系里被什么伤过（**等对方主动说**）
- [ ] 对方在感情里最怕什么
- [ ] 对方想要这段关系是什么样
- [ ] 对方希望吵架时怎么处理
- **解锁信号：** 你俩能平静地聊"我们之间"而不紧张。

## 三、理论背书 A —— Gottman 内心地图（Love Maps）四层

> Gottman 研究：对彼此有详细"内心地图"的伴侣，幸福长久的概率高出约 **90%**。
> "内心地图"= 你脑子里储存的、关于对方内心世界的所有信息。要采集的内容可分成四层（对应上面的 L0–L5）：

| 层 | 内容 | 对应 Level |
|---|---|---|
| 第 1 层 · 日常表层 | 朋友、爱好、今天干了啥、爱吃什么、讨厌谁 | L0–L1 |
| 第 2 层 · 当下世界 | 压力源、烦心事、在期待什么、近期状态 | L1–L2 |
| 第 3 层 · 来历与内心 | 成长经历、塑造的事、骄傲与伤 | L3 |
| 第 4 层 · 价值观与梦想 | 在乎什么、世界观、想要的未来、底线 | L4–L5 |

**维护要点：** 人会变，地图要**定期更新**——隔段时间坐下来"对齐"一次近况。

## 四、理论背书 B —— Aron"让人相爱的 36 个问题"（问题库）

> 1997 年 Arthur Aron 的实验：陌生人按由浅入深的 36 问一来一回，能显著拉近亲密感。
> **精髓不在题，在规律：① 由浅入深 ② 一来一回、双向袒露 ③ 别太快太多。**（正好对应协议规则 1、2、4）
> 用法：当成"问题库"，按当前 Level 从对应 Set 里挑，**轮流问、你也答**。

### Set I（轻 · 对应 L0–L2）
1. 全世界任选，你最想请谁来吃饭？
2. 你想出名吗？以什么方式？
3. 打电话前，你会先打草稿想好怎么说吗？为什么？
4. 对你来说，"完美的一天"是什么样的？
5. 你上次唱歌给自己听是什么时候？给别人听呢？
6. 如果能活到 90 岁，且后 60 年能保持 30 岁的"身体"或"头脑"，你选哪个？
7. 关于自己会怎么死，你有没有过某种隐隐的预感？
8. 说出三件你和对方看起来有的共同点。
9. 你人生中最感激的是什么？
10. 如果能改变成长过程中的任何一件事，你想改什么？
11. 花四分钟，尽可能详细地把你的人生故事讲给对方听。
12. 如果明天醒来能获得一种品质或能力，你想要什么？

### Set II（中 · 对应 L2–L4）
13. 如果有个水晶球能告诉你关于自己 / 人生 / 未来的真相，你想知道什么？
14. 有没有什么你梦想了很久却一直没做的事？为什么没做？
15. 你这辈子最大的成就是什么？
16. 你在友情里最看重什么？
17. 你最珍贵的回忆是什么？
18. 你最糟糕的回忆是什么？*(敏感，pull-only)*
19. 如果知道一年后会突然死去，你会改变现在的活法吗？为什么？
20. 友情对你意味着什么？
21. 爱与亲密在你生命里扮演什么角色？
22. 轮流说出你眼中对方的一个优点，各说五个。
23. 你的家庭亲密温暖吗？你觉得自己的童年比大多数人幸福吗？
24. 你和母亲的关系怎么样？*(敏感)*

### Set III（深 · 对应 L4–L5，需高信任）
25. 各说三句真实的"我们"句子，比如"我们此刻都在这个房间里，感到……"
26. 补全这句："我希望有个人能和我一起分享……"
27. 如果要和对方成为密友，有什么是你觉得 ta 一定要知道的？
28. 诚实地告诉对方你喜欢 ta 的地方（说些对刚认识的人不会说的）。
29. 跟对方分享一件你人生中尴尬的事。
30. 你上次在别人面前哭是什么时候？独自哭呢？*(敏感)*
31. 告诉对方一个你现在就已经喜欢 ta 的地方。
32. 有什么事，在你看来严肃到不能拿来开玩笑？
33. 如果今晚就死、再没机会跟任何人说话，你最后悔没告诉谁什么？为什么还没说？
34. 你的房子着火，救出人和宠物后还能再冲进去拿一样东西，你拿什么？为什么？
35. 家里所有人中，谁的离世会让你最难受？为什么？*(敏感)*
36. 说一个你的私人难题，问对方会怎么处理；再请对方复述一下"你看起来对这件事是什么感受"。

> ⚠️ 标注 *(敏感)* 的题，在信任未建立时**不要主动问**，留给对方自己提起。
